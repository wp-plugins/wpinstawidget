WPInstaWidget plugin for display of your account Instagram in Wordpress

== Description ==

Display Your Instagram Profile into WordPress Site. 3 unique templates. Authorization in the Instagram isn't necessary. Use only the login!

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'WPInstaWidget'
3. Activate WPInstaWidget from your Plugins page.
4. Visit Options > WPInstaWidget.
5. Enter your login into an Instagram in the field of "Your instagram account login" and press the button "Save Changes"
6. In the block "Shortcode options" choose a type of display, number of the output photos and set parameters of width and height if it is necessary.
7. Press "Show Result" button for a preview.
8. Copy a short-code and you can publish Instagram provide widget in any Page or Post in your WordPress sites.

= From WordPress.org =

1. Download WPInstaWidget.
2. Upload the 'WPInstaWidget' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate WPInstaWidget from your Plugins page.
4. Visit Options > WPInstaWidget.
5. Enter your login into an Instagram in the field of "Your instagram account login" and press the button "Save Changes"
6. In the block "Shortcode options" choose a type of display, number of the output photos and set parameters of width and height if it is necessary.
7. Press "Show Result" button for a preview.
8. Copy a short-code and you can publish Instagram provide widget in any Page or Post in your WordPress sites.
